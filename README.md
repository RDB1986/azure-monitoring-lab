# Azure Monitoring & Alerting Lab

## 📌 Overview

This project demonstrates how to deploy a virtual machine in Microsoft Azure, configure monitoring, and create alert rules based on CPU utilization.

---

## ⚙️ What I Built

* Deployed Windows Server VM in Azure
* Configured RDP access
* Set up Azure Monitor metrics
* Created CPU alert rule
* Simulated CPU load using PowerShell

---

## 🚨 Alert Configuration

* Metric: Percentage CPU
* Threshold: Greater than 15%
* Aggregation: Average
* Frequency: 1 minute

---

## 🧪 Testing

* Generated CPU load manually
* Observed spike in Azure Monitor
* Successfully triggered alert

---

## 📊 Results

* Alert fired when CPU exceeded threshold
* Verified in Azure portal

---

## 🧠 What I Learned

* Azure Monitor alert setup
* CPU metric behavior and delay
* Importance of threshold tuning
* Basic cloud troubleshooting
